## 1. Что такое FlashCard Studio?

**FlashCard Studio** — это настольное приложение для создания и изучения карточек-флэшкарт. Приложение помогает запоминать любую информацию эффективнее, используя метод **интервального повторения (Spaced Repetition)**.

### Как работает интервальное повторение?

Идея проста: карточки, которые вы хорошо знаете, показываются реже, а карточки, которые вызывают затруднение, — чаще. Так вы тратите время именно на то, что ещё не запомнили, а не повторяете то, что уже знаете.

Приложение автоматически рассчитывает, когда нужно снова показать каждую карточку, и напоминает её в нужный момент.

### Для чего можно использовать?

- Изучение иностранных слов и выражений
- Подготовка к экзаменам и тестам
- Запоминание дат, формул, терминов
- Повторение любого учебного материала

## 2. Установка

### Системные требования

- Операционная система: Windows 10 или Windows 11 (64-бит)
- Оперативная память: 100 МБ (приложение)
- Дисковое пространство: ~80 МБ

### Установка из установщика

1. Запустите файл `FlashCard Studio-1.0.0-win64.exe`.
2. Следуйте инструкциям мастера установки.
3. После установки на рабочем столе и в меню «Пуск» появится ярлык **FlashCard Studio**.
4. Файлы проектов с расширением `.flcr` будут автоматически открываться двойным кликом.

## 3. Первый запуск

При запуске приложения открывается **стартовый экран**.

```
┌──────────────────┬─────────────────────────────────┐
│                  │  🕒 Недавние проекты             │
│  FlashCard       │                                  │
│  Studio          │  (список последних открытых      │
│                  │   файлов .flcr)                  │
│  📄 Создать      │                                  │
│     проект       │                                  │
│                  │                                  │
│  📂 Открыть      │                                  │
│     проект       │                                  │
└──────────────────┴─────────────────────────────────┘
```

- **Левая панель** содержит кнопки для создания нового проекта или открытия существующего.
- **Правая панель** показывает список недавно открытых проектов. Двойной клик по записи мгновенно открывает его.

## 4. Создание и открытие проекта

### Создание нового проекта

1. Нажмите кнопку **«Создать проект»** на стартовом экране  
   *или* выберите **Файл → Создать** (`Ctrl+N`) в главном окне.
2. В диалоге сохранения выберите папку и введите имя файла. Файл будет сохранён с расширением `.flcr`.
3. Приложение перейдёт в главное окно — проект создан и готов к работе.

### Открытие существующего проекта

- Нажмите **«Открыть проект»** на стартовом экране  
  *или* выберите **Файл → Открыть** (`Ctrl+O`).
- В диалоге найдите и выберите файл `.flcr`.

Также можно открыть файл `.flcr` двойным кликом в Проводнике Windows — приложение запустится автоматически.

### Закрытие проекта

- Выберите **Файл → Закрыть проект**.
- Приложение вернётся на стартовый экран. Если есть несохранённые изменения, программа спросит, сохранить ли их.

## 5. Работа с колодами

**Колода** — это именованная группа карточек. Например, «Английские глаголы», «История России», «Формулы физики».

Список всех колод находится на **боковой панели** слева.

### Создание колоды

Нажмите кнопку **➕** в верхней части боковой панели. Введите название новой колоды и нажмите Enter.

### Поиск колоды

Введите текст в поле **🔍 Поиск колоды...** — список отфильтруется в реальном времени.

### Действия с колодой

При наведении курсора на любую колоду справа появляются две кнопки:

| Кнопка | Действие |
|--------|----------|
| ✏️ | Открыть колоду в **редакторе** (добавить/изменить карточки) |
| ▶️ | Начать **сессию обучения** по этой колоде |

### Удаление колоды

- Нажмите на колоду, чтобы выделить её, затем нажмите кнопку **🗑️** на боковой панели.
- Подтвердите удаление в диалоге. Все карточки в колоде будут удалены вместе с ней.

## 6. Редактирование карточек

Нажмите кнопку ✏️ рядом с колодой — откроется **экран редактирования**.

```
┌──────────────────────────────────────────────────────────────┐
│  Название колоды        [➕ Добавить]  [🗑️ Удалить]  Карточек: 12 │
├──────────────────────────────────────────────────────────────┤
│  Вопрос                    │  Ответ                          │
│  ─────────────────────────────────────────────────────────   │
│  hola                      │  hello                          │
│  guten Morgen              │  доброе утро                    │
│  ▶ (выбранная строка)      │                                 │
├──────────────────────────────────────────────────────────────┤
│  [Ж] [К] [Ч]               │  [Ж] [К] [Ч]                   │
│  ┌──────────────────────┐  │  ┌──────────────────────────┐   │
│  │ Лицевая сторона      │  │  │ Обратная сторона         │   │
│  │ (поле ввода вопроса) │  │  │ (поле ввода ответа)      │   │
│  └──────────────────────┘  │  └──────────────────────────┘   │
└──────────────────────────────────────────────────────────────┘
```

### Добавление карточки

1. Нажмите кнопку **➕ Добавить** или используйте `Ctrl+N`.
2. В таблице появится новая пустая строка, которая будет выбрана автоматически.
3. В нижней части экрана откроются два поля ввода — введите **вопрос** (лицевая сторона) и **ответ** (обратная сторона).

### Редактирование карточки

1. Кликните по строке в таблице, чтобы выбрать карточку.
2. В нижней части появятся поля с содержимым карточки.
3. Внесите изменения — они автоматически сохранятся в памяти через 500 мс после паузы в наборе текста.

### Форматирование текста

Над каждым полем ввода есть панель форматирования:

| Кнопка | Действие | Горячая клавиша |
|--------|----------|-----------------|
| **Ж** | Жирный текст | `Ctrl+B` |
| *К* | Курсив | `Ctrl+I` |
| <u>Ч</u> | Подчёркивание | `Ctrl+U` |

Выделите текст и нажмите нужную кнопку или сочетание клавиш.

### Удаление карточки

Выделите карточку в таблице и нажмите кнопку **🗑️ Удалить** (или клавишу `Del`). Действие запрашивает подтверждение.

## 7. Режим обучения

Нажмите кнопку **▶️** рядом с колодой на боковой панели — откроется **экран обучения**.

Приложение автоматически выберет карточки, которые нужно повторить сегодня (те, у которых пришло время повторения).

### Ход сессии

**Шаг 1 — Вопрос**

На экране отображается **лицевая сторона** карточки. Прочитайте её и постарайтесь вспомнить ответ.

```
┌─────────────────────────────┐
│                             │
│         hola                │
│                             │
└─────────────────────────────┘

         [👁️ Показать ответ]
```

Нажмите кнопку **«Показать ответ»** или клавишу `Пробел`.

**Шаг 2 — Оценка**

Откроется **обратная сторона** карточки. Оцените, насколько хорошо вы вспомнили ответ, выбрав одну из четырёх кнопок:

```
┌─────────────────────────────┐
│         hola                │
│  ──────────────────────── │
│         hello               │
└─────────────────────────────┘

 [✖️ Снова]  [😟 Трудно]  [🙂 Хорошо]  [⭐ Легко]
   1 д.        4 д.         15 д.        28 д.
```

Под каждой кнопкой показано, **через сколько дней** карточка появится снова при выборе этого ответа.

### Описание кнопок оценки

| Кнопка | Цвет | Когда нажимать |
|--------|------|----------------|
| **Снова** | Красная | Вы совсем не помните ответ. Карточка вернётся сегодня снова. |
| **Трудно** | Оранжевая | Вспомнили с трудом или с ошибками. Карточка придёт скоро. |
| **Хорошо** | Синяя | Вспомнили правильно, но не мгновенно. Стандартный интервал. |
| **Легко** | Зелёная | Вспомнили сразу и без усилий. Карточка придёт намного позже. |

> **Совет:** Старайтесь оценивать честно! Программа запомнит ваш выбор и подберёт оптимальный момент для следующего повторения.

### Прогресс сессии

В верхней части экрана отображается полоса прогресса и счётчик оставшихся карточек.

### Завершение сессии

Когда все карточки пройдены, появляется экран с сообщением:

```
        🎉
   На сегодня всё!
```

Нажмите кнопку «Назад» в верхнем левом углу, чтобы вернуться к списку колод.

### Что если карточки не показываются?

Если при нажатии ▶️ не появляется ни одной карточки, значит на сегодня нет карточек для повторения — все интервалы ещё не истекли. Возвращайтесь завтра!

## 8. Сохранение и автосохранение

### Ручное сохранение

- **Сохранить** (`Ctrl+S`) — сохраняет проект в текущий файл.
- **Сохранить как** — сохраняет проект в новый файл `.flcr` (исходный файл остаётся без изменений).

### Автосохранение

Приложение **автоматически сохраняет** проект каждые 3 минуты, если в нём есть несохранённые изменения. Вы можете продолжать работу — данные не пропадут.

### Индикатор изменений

В заголовке окна отображается `*` (звёздочка) рядом с именем проекта, если есть несохранённые изменения. После сохранения звёздочка исчезает.

## 9. Темы оформления

Приложение поддерживает два варианта оформления:

- **Светлая тема** — белый фон, подходит для работы при хорошем освещении.
- **Тёмная тема** — тёмный фон, снижает нагрузку на глаза в сумерках или ночью.

Для переключения выберите **Вид → Светлая тема** или **Вид → Тёмная тема**.

Выбор сохраняется и будет применяться при каждом запуске автоматически.

## 11. Часто задаваемые вопросы

**Где хранятся мои данные?**  
Все данные хранятся в файле `.flcr`, который вы указали при создании проекта. Приложение не отправляет данные в интернет.

**Можно ли перенести проект на другой компьютер?**  
Да. Просто скопируйте файл `.flcr` на другой компьютер и откройте его через **Файл → Открыть** или двойным кликом в Проводнике (при условии, что FlashCard Studio установлен).

**Почему некоторые карточки не появляются в сессии обучения?**  
Это нормально. Если вы недавно оценили карточку как «Хорошо» или «Легко», программа запланировала следующее повторение на несколько дней вперёд. Возвращайтесь в следующий раз — карточка появится в нужный момент.

**Что произойдёт, если я нажму «Снова»?**  
Карточка вернётся в конец очереди текущей сессии — вы увидите её снова сегодня. Её интервал также будет сброшен до 1 дня.

**Можно ли добавить форматирование (жирный шрифт, курсив) в карточки?**  
Да. В редакторе выделите нужный текст и нажмите кнопки форматирования **Ж**, **К**, **Ч** над полем ввода или используйте горячие клавиши `Ctrl+B`, `Ctrl+I`, `Ctrl+U`.

**Как удалить колоду?**  
Наведите курсор на колоду в боковой панели. Появятся кнопки — нажмите на иконку удаления. Подтвердите действие в появившемся диалоге.

**Что означает число под кнопками «Снова / Трудно / Хорошо / Легко»?**  
Это предварительный просмотр — через сколько дней карточка появится снова, если выбрать эту оценку. Например, «15 д.» означает, что карточка вернётся через 15 дней.

**Приложение закрылось случайно — я потерял данные?**  
Нет. Автосохранение срабатывает каждые 3 минуты. Самое большее, что может потеряться — это изменения, сделанные менее чем за 3 минуты до внезапного закрытия. Для надёжности используйте `Ctrl+S` почаще.
